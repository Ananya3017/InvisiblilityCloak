# 🪄 Invisibility Cloak Project (OpenCV)

This project creates a **Harry Potter-style invisibility cloak** using Python and OpenCV.  
It replaces the color of the cloak with the background, creating the illusion of invisibility.

---

## 🎯 Features

- **Multiple versions included**:
  - `cloak_red.py` → Basic version (works with red cloak)
  - `cloak_custom.py` → Enhanced version (choose any cloak color)
- **Multi-color cloak support** (red, blue, green, yellow, orange, purple)
- **Smooth edges** with Gaussian blur for realistic effect
- **Keyboard shortcuts** for changing cloak colors and recapturing background
- **Background recapture** at any time

---
